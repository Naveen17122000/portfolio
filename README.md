## Tech Stack

This site is intentionally framework-free — a single static HTML file with no build step required.

| Layer | Technology |
|---|---|
| Structure | HTML5 |
| Styling | Vanilla CSS (custom properties for theming) |
| Interactivity | Vanilla JavaScript (`IntersectionObserver`, Canvas API, `requestAnimationFrame`) |
| Fonts | [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) · [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) (Google Fonts) |
| Hosting | GitHub Pages |

No React, no Tailwind, no npm install, no build process. Open `index.html` in any browser and it works.

## Features

- 🎨 API-console themed design — hero renders as a live terminal session
- ✨ Staggered entrance animations on load
- 📜 Scroll-triggered reveal animations for each section
- 🧭 Active-section nav highlighting while scrolling
- 📊 Animated skill proficiency bars
- 🌌 Ambient particle-canvas background
- 📱 Fully responsive (mobile, tablet, desktop)
- ♿ Respects `prefers-reduced-motion` for accessibility

## Running Locally

No installation needed.

```bash
git clone https://github.com/Naveen17122000/Naveen17122000.github.io.git
cd Naveen17122000.github.io
```

Then just open `index.html` directly in your browser, or serve it locally:

```bash
python3 -m http.server 8000
# visit http://localhost:8000
```

## Deployment

This repo is deployed via **GitHub Pages**, set to serve from the `main` branch root. Any push to `main` updates the live site within a minute or two.
